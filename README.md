# UIDevice-Hardware
相关技术贴链接：https://www.jianshu.com/p/308a4a85e114

觉得用着还行记得给个star~ ：）

使用方法如下：
```
#import "UIDevice+Hardware.h" //需要import头文件

UIDevice *device = [UIDevice currentDevice];
NSLog(@"%@", [device generation]); //输出
```
